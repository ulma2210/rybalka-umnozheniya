# Рыбалка умножения

Небольшая локальная браузерная игра для тренировки таблицы умножения.

## Что внутри

- короткие раунды на время
- выбор ответа нажатием на рыбку
- ускорение игры по мере роста счёта
- сохранение лучшего результата в браузере
- адаптация под компьютер и телефон

## Как запустить

1. Откройте `index.html` в браузере.
2. Или запустите простой локальный сервер в папке проекта:

```bash
python3 -m http.server
```

После этого откройте адрес, который покажет терминал.

## Файлы проекта

- `index.html` — структура экранов
- `style.css` — стили и анимации
- `script.js` — логика игры
